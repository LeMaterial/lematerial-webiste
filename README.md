## How to contribute

### Install Requirements:
- hugo (https://gohugo.io/installation/)
- npm (https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

### Install dependencies:

```bash
npm install
```

### Start locally:
```bash
npm run dev
```

content (doc) is at content/docs

### Deploy the web

```bash
npm run build
npm run deploy
```
